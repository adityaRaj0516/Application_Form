# Professional Candidate Portal

A responsive, modern candidate application portal built with HTML, CSS, and Vanilla JavaScript. 
This project demonstrates a clean UI/UX for collecting applicant data, including a custom tag-input system for skills and file upload handling.

# Features

- **Responsive Design:** Fully responsive layout using Flexbox and CSS Grid.
- **Corporate UI:** Clean styling with Inter font and a professional color palette.
- **Custom Form Controls:**
  - **Tag Input:** Interactive skill tagging system (add via Enter/Comma, remove via click).
  - **File Upload:** Styled upload button with dynamic file name display.
- **Client-Side Validation:** Real-time error handling for all required fields.
- **Mock Submission:** Simulates an asynchronous API call with a loading state and success summary view.
